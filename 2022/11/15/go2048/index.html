创建应用

进入go2048目录，初始化工程

npm install

运行：

npm run dev

导入依赖

npm i -D @types/node

npm i -D unocss

npm i -D @iconify-json/carbon

npm i -D sass

npm i @vueuse/core

npm i guid-typescript
